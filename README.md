## これはなに

react+redux プロジェクトのボイラープレート。create-react-app で生成されたコードがベース

## 使い方

このレポジトリをローカルにクローンしてディレクトリに移動し`npm install`

* `npm start`で開発用サーバー起動`localhost:3000`でページにアクセス
* `npm test`でテスト実行
* `npm build`でプロダクション用ビルドを生成
